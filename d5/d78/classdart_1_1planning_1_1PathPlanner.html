Performs bidirectional RRT with the given options. NOTE This algorithm has several different popular implementations. The implementation in the kinodynamic paper (1999) by LaValle and Kuffner extend the two RRTs towards a common random configurations whereas here, first, start rrt extends towards a random node and creates some node N. Afterwards, the second rrt extends towards the node N and they continue swapping roles.
